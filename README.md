
## 阅读步骤
- 根目录下的入口文件即源码入口
- 选中部分代码全局搜索即可找到对应源码


## 快捷键
| 方法名                      | 说明         | 快捷键         | 推荐增加 |
| --------------------------- | ------------ | -------------- | -------- |
| Go to Declaration or Usages | 进入代码调用 | Ctrl + B       | Alt + E  |
| Back                        | 返回调用     | Ctrl + Alt + ← | Alt + Q  |
